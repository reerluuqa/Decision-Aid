Palpitations Assessment

URGENT: Palpitations + Syncope = High Risk VT

Immediate 12-lead ECG, cardiac monitor, IV access. Check BP, O2 sat. Prepare for cardioversion if unstable.

Red Flags - Immediate Action Required

CRITICAL

High-Risk Clinical Features

Must Exclude
  • Syncope with palpitations
  • Structural heart disease
  • Family Hx sudden death
  • Prior MI / Cardiomyopathy
Immediate Actions
  • 12-lead ECG NOW
  • Continuous monitoring
  • IV access
  • Echo if LV dysfunction suspected
Sustained tachycardia >30 sec = Emergency triage O2 sat, BP monitoring, urine output if unstable

Associated Features
  • Polyuria: SVT → atrial distension → ANP release
  • Exercise-induced: RVOT VT, Long QT, inappropriate sinus tachy
  • Awakening: Consider sleep apnoea + AF
  • Position-related: AVNRT when bending over

Physical Examination Clues

Finding
  • Irregular pulse
  • Cannon A waves (JVP)
  • Displaced apex
  • Harsh systolic murmur ↑Valsalva
  • Mid-systolic click
  • S3 gallop
Suggests
  • AF
  • AV dissociation (VT)
  • Dilated cardiomyopathy
  • HOCM
  • Mitral valve prolapse
  • Heart failure
Check for hyperthyroid signs: Tremor, tachycardia, lid lag, exophthalmos, warm skin

AVNRT / AVRT (SVT)

Features
  • Abrupt onset/offset
  • "Machine-like"
  • Pounding in neck
  • Valsalva may terminate
  • Age 20-30s typical
ECG Clues
  • Narrow QRS (usually)
  • Pseudo-S in inferior leads
  • Pseudo-R' in V1
  • Regular rate 150-250
