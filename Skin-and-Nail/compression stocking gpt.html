Compression Stockings — One‑Page Clinical Algorithm

Adults ≥16 years • Clean, fast reference for consultations • Summary Page

1 · Pre‑assessment

History & Exam

  • Trauma/infection; comorbidities; meds (ACEi, steroids, immunosuppressants); family/PMH of venous disease; prior DVT.
  • Look for varicose veins, venous eczema, lipodermatosclerosis, atrophie blanche, healed/active ulcers; assess oedema and non‑venous causes.
Exclude arterial disease first:
ABPI < 0.8 → Contraindicated · Refer vascular 0.8–1.3 → Safe to start > 1.3 → Calcified arteries suspected · Avoid · Refer

Contraindications

Peripheral arterial disease / bypass graft Peripheral neuropathy / sensory loss Fragile skin, dermatitis, gangrene, recent graft Allergy to material Severe leg oedema Major limb deformity Acute infection foot/leg Suspected acute DVT Suspected skin cancer
High oedema preventing ABPI? Consider up to 20 mmHg (Class 1) to reduce oedema and re‑attempt ABPI ~1 week later if no other contraindications.
2 · Select Compression Class (highest tolerated)

Condition → Class

  • Varicose veins: Class 1–2 (per severity & tolerance).
  • Venous eczema / lipodermatosclerosis / atrophie blanche / healed venous ulcer: Start Class 2 → escalate to Class 3 if inadequate; drop to Class 1 if intolerance.
  • Superficial vein thrombosis: Class 2 (alternatives: Class 1 or travel socks if painful).
  • After proximal DVT: Do not prescribe for PTS/VTE recurrence prevention.
British standard (ankle pressure): Class 1 14–17 mmHg · Class 2 18–24 mmHg · Class 3 25–35 mmHg.
3 · Length & Toe

Choose Practical Options

  • Length: Below‑knee = default (ulcer‑recurrence prevention; swelling below knee). Thigh‑length if severe varicosities or oedema above knee.
  • Toe: Open‑toe if fungal infection, arthritic/clawed toes, podiatry, sock over‑wear, or long foot vs calf; cooler but may ride up / cause toe oedema. Closed‑toe if toe oedema or foot‑joint pain.
4 · Measurement & Fit

Measure Accurately (each leg)

  • Measure on bare skin; consider skin marker for consistency.
  • For below‑knee: ankle (narrowest above malleoli), calf (widest), foot length (if closed‑toe).
  • For thigh‑length: add thigh (widest).
  • Take morning measurements if oedematous; compression bandaging first if severely swollen.
  • Use made‑to‑measure if deformity/poor fit; pharmacist can endorse prescription.
5 · Prescribing & Use

Write a Clear Prescription

  • Specify class, length, toe style, size; add “made to measure” if applicable.
  • Apply in morning; remove at bedtime (can be worn continuously up to 7 days if necessary).
  • Replace every 3–6 months; supply ≥2 stockings/pairs; re‑measure on replacement.
  • Care: hand wash ≈40°C; dry away from direct heat.
6 · Review & Adherence

Follow‑up

  • Review every 3–6 months: reassess indication, fit, wear, replacement; re‑measure legs.
  • Repeat Doppler every 6–12 months (earlier if indicated / per local policy).

Boost Adherence

  • Explain benefits; demonstrate donning/doffing.
  • Consider application aids (e.g., ActiGlide®).
  • Use emollient while off; adjust compression or length for tolerance.
Summary

Bottom line

Exclude arterial insufficiency → choose the highest tolerated class → match length & toe to need → ensure precise measurement & fit → schedule regular replacement and review with adherence support.
