Headache – One-Page Quick Reference

CKS 2022 • Adults ≥16 y • Primary-care triage

30 sEmergency Screen – RED FLAGS

  • Thunderclap (max intensity ≤5 min)
  • New severe headache >50 y
  • Progressive / worsening pattern
  • Fever + neck stiffness / seizure / focal neuro / papilloedema
  • Immunosuppression, cancer, pregnancy, recent head trauma

Any flag → same-day A&E / neuro referral

AskLook for

Examine: vitals, ENT, neck, neuro, fundi.

Primary Headaches

TypeKey Features
Migraine4–72 h, unilateral, throbbing, N/V, photo/phonophobia
Tension30 min–7 d, bilateral, pressing, mild-mod, no nausea
Cluster15–180 min, severe orbital, autonomic, restlessness
Med-overuse≥15 days/mo for >3 mo, overusing abortives

When to Refer / Image

  • Red flags → immediate
  • Atypical aura → routine neuro
  • GCA suspected → same-day ESR/CRP + urgent biopsy
  • No imaging for typical migraine/tension

Workflow Summary

  1. Screen red flags (30 s)
  2. Apply ICHD criteria (60 s)
  3. Diary 8 w if uncertain, no flags
  4. Safety-net & follow-up
