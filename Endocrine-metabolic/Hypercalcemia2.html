Hypercalcaemia Quick Aid

Fast decision tool for GP consults • Corrected Calcium (Albumin adjusted)

Red Flags

Symptoms of Hypercalcaemia

  • Thirst & polyuria/frequency
  • Constipation
  • Nausea/vomiting
  • Drowsiness (late sign - urgent)

Primary Hyperparathyroidism Features

  • Bone pathology: Osteoporosis, fragility fracture
  • Renal pathology: Renal stone

Consider Checking if Chronic Non-Specific:

Cancer Association

Check if known/suspected:

  • Lung cancer
  • Myeloma
  • Other malignancy associated with hypercalcaemia

Mild-Moderate Hypercalcaemia

2.60–2.99 mmol/l

Any symptoms of hypercalcaemia?
If YES: Urgent referral/Admission
If NO symptoms: Repeat corrected calcium

Note: 90% due to primary hyperparathyroidism or malignancy

High-Normal Range

2.50–2.59 mmol/l

AND any features of primary hyperparathyroidism?
(bone/renal pathology)
If YES: Repeat corrected calcium

If normal range AND no features → Routine monitoring

Repeat Testing Protocol

If on 2 separate occasions:

  • Corrected calcium ≥2.6 mmol/l
  • OR Calcium 2.50–2.59 AND features of primary hyperparathyroidism

→ Arrange PTH and repeat corrected calcium

PTH Interpretation Guide

Click cards for details

PTH Low

Below reference range
Non-PTH dependent hypercalcaemia

Causes:

  • Cancer (lung, myeloma, bony mets)
  • Thyrotoxicosis
  • Vitamin D toxicity
  • Sarcoidosis
  • Thiazides
  • Acromegaly
  • Phaeochromocytoma

PTH Mid-Low + Low Ca

Below midpoint
(but within range) AND Ca <2.6
Primary hyperparathyroidism unlikely

PTH Mid-Low + High Ca

Below midpoint
AND Ca ≥2.6
Cannot rule out primary hyperparathyroidism

Seek endocrine advice

PTH Raised

Above midpoint or raised
Likely primary hyperparathyroidism

Seek endocrine advice (referral/telephone)

Other causes: Tertiary hyperparathyroidism (renal), Familial (MEN1, FHH), Lithium

Other Tests

NICE says NO

Do not check Vitamin D or other tests when assessing for primary hyperparathyroidism

Local Guidance

Often suggests checking Vitamin D to assess underlying cause

Useful Tests for Underlying Cause:

  • Bone profile & U&Es
  • TFTs
  • Myeloma screen (if indicated)
  • Prolactin (if <45y - for familial causes)
  • Calcium excretion index (urine)

Immediate Actions

If on Thiazide diuretic: STOP
If excess dietary calcium: REDUCE
If Vitamin D low: Do NOT delay referral

Non-PTH Dependent Causes

When PTH is low (suppressed)

Familial/Hereditary Causes

Consider if PTH raised and:

  • Patient <45 years (check prolactin)
  • Family history
  • Urine calcium excretion index
Types: MEN1, Familial hypocalciuric hypercalcaemia (FHH)

Renal Disease

Consider tertiary hyperparathyroidism in patients with chronic renal disease
