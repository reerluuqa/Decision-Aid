Pruritus Vulvae – Quick Consult Guide

One-page reference algorithm • Last updated Oct 2022 • NICE CKS

1. First 30-second Triage

  • Confirm itch, not pain
  • Pregnant? → See Itch in pregnancy CKS
  • Target group: adult women ≥18 y

2. 60-Second History Snaps

QuestionRed-flag Clues
Location / DurationExtragenital → generalized itch
Acute onsetAllergy / candida
Nocturnal itchThreadworm
Discharge / odourCandida, BV, STI
Continence issuesContact dermatitis
New lump/ulcer/bleedingURGENT 2-week referral

3. 30-Second Examination

  • Consent + chaperone
  • Inspect vulva: colour, fissures, lichenification, ulcers, warts, atrophy
  • Speculum if discharge → high vaginal swab
  • Extragenital: elbows/knees (psoriasis), mouth (lichen planus)

4. Instant Bedside Tests

  • Discharge / infection suspected → high vaginal swab ± STI screen
  • Generalized itch → FBC, ferritin, TFT, renal, LFT, HbA1c

5. Management Decision Tree

graph TD A[Pruritus vulvae] --> B{Known cause?} B -->|YES| C[Treat specific cause] B -->|NO / Unclear| D[Empiric & symptomatic] C --> C1[Candida → treat] C --> C2[Contact dermatitis → remove irritant + mild–potent TCS ointment 7–10 d] C --> C3[Lichen sclerosus/planus → urgent referral] C --> C4[Neoplasm → 2-week referral] D --> D1[Emollient as soap substitute] D --> D2[Low-potency TCS ointment 1–2 w trial] D --> D3[± antifungal/antibacterial combo if infection suspected] D --> D4[Night sedation: Hydroxyzine or Amitriptyline 10 mg] D5[Self-care sheet] D --> E{Improved?} E -->|NO| F[Refer dermatology / gynae / vulval clinic]

6. Referral Quick-List

  • 2-week referral unexplained lump / ulcer / bleeding
  • 2-week referral suspected VIN / lichen sclerosus / planus
  • Persistent symptoms ≥4–6 w despite primary care treatment

7. Core Self-Care Script

  • Shower daily with emollient as soap substitute
  • Dab dry gently; cotton underwear
  • Avoid soap, perfume, wipes, shaving, tight synthetic clothing
  • Cool packs for itch; keep nails short

Pocket Prescribing

  • Emollient: apply QDS + as soap substitute
  • TCS ointment (short burst):
    • Mild: Hydrocortisone 1 %
    • Moderate-severe: Mometasone or Betamethasone 0.025 %
  • Night itch: Hydroxyzine 25 mg (avoid if >65 y / QT risk) or Amitriptyline 10 mg
